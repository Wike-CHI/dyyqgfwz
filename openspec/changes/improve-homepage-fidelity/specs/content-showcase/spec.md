## ADDED Requirements

### Requirement: Masonry Grid Component
The system SHALL provide a `MasonryGrid` component to display "What's New" content in a dense, visually rich layout.

#### Scenario: Grid Rendering
- **WHEN** the `MasonryGrid` is rendered with items
- **THEN** it should display items in a multi-column layout that adapts to screen size

### Requirement: Video Section Component
The system SHALL provide a `VideoSection` component for "Tree House in Motion" content.

#### Scenario: Video Section Rendering
- **WHEN** the `VideoSection` is rendered
- **THEN** it should display a video player or thumbnail with a play button

### Requirement: Feature Grid Component
The system SHALL provide a `FeatureGrid` component for the "Grain to Glass" section.

#### Scenario: Feature Grid Rendering
- **WHEN** the `FeatureGrid` is rendered
- **THEN** it should display feature blocks (image + text) in a grid layout

### Requirement: Quote Section
The system SHALL provide a simple text-based section for brand quotes.

#### Scenario: Quote Rendering
- **WHEN** the quote section is rendered
- **THEN** it should display centered text with specific typography
