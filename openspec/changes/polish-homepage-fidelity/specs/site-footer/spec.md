## MODIFIED Requirements

### Requirement: Site Footer
The system SHALL provide a global `Footer` component containing links, copyright information, and social media icons, with updated styling.

#### Scenario: Footer Rendering
- **WHEN** the `Footer` is rendered
- **THEN** it should be visible at the bottom of the page
- **THEN** it should contain navigation links and copyright text
- **THEN** it should have a dark background color consistent with the design system
